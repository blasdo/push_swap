# ft\_isdigit
## Sintaxis y explicación
### Prototipo
```c
int	ft_isdigit(int c)
```
### Descripción
ft\_isalpha comprueba que el entero 'c' sea casteable a unsigned char y que ese unsigned char sea un dígit(de '0' a '9') o, para eso hace uso de la tabla ascii.
## Retorno
Devuelve 1 si el entero 'c' está dentro de los valores de un unsigned char y su valor en la tabla ascii es el de un dígito
